## Installation :
- Cloner le repo
    * `git clone https://github.com/Niilyx/sudoku-nsi.git` via HTTPS
    * `git clone git@github.com:Niilyx/sudoku-nsi.git` via SSH
- Ouvrez le fichier [index.html](/index.html) avec votre navigateur
